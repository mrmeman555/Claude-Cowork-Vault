## What ML OS Is

A **Meta-Language Operating System** — a layered framework for composing AI agent system prompts and managing the context they operate on. Five layers:

| Layer | OS Analogy | Role |
|-------|-----------|------|
| S1 ML OS Core | Kernel/Firmware | Immutable identity, constants, boot sequence |
| S2 AI Schema | CPU/Runtime | Reasoning protocol, interaction style, output behavior |
| S3 Scenario | App Layer | Role definition, objectives, evaluation criteria |
| S4 Sources | Storage/Input | Documents, datasets, transcripts |
| S5 Outputs | Display/Results | Structured deliverables |

Key concepts:
- **Bridge Pattern** (from Notion Bridge Architect): Brain <-> Bridge Agent <-> Execution Platform
- **Master Context Document**: Living, self-updating document tracking workspace state
- **Context Packs**: Curated sets of files + agent prompts for specific purposes
- **Template Composition**: Kernel + Schema + Scenario YAML -> full system prompt
